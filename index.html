Cours suivi

Concepts clés

Partie 1 - Découvrez le web et ses origines

Web

Le web est un ensemble de pages et de sites accessibles sur Internet grâce à unnavigateur

Internet

Internet est le grand réseau mondial qui relie entre eux des millions d’ordinateurs pour échanger des informations

Cloud

Le cloud désigne une technologie où des données (fichiers, programmes..etc) sont stockés sur des serveurs accessibles à distance, et non uniquement sur un ordinateur local

W3C

Le W3C est un groupe international qui fixe les règles pour que les sites web fonctionnent bien partout et pour tout le monde

Partie 2 - Distinguez les langages du Web

Langages client

Ce sont des langages, comme HTML, CSS ou JavaScript, qui s’exécutent dans le navigateur de l’utilisateur pour afficher et rendre interactives les pages web

Navigateurs

Un navigateur est un logiciel client qui se connecte à des serveurs web pour télécharger, interpréter et afficher les pages web à l’aide de standards comme HTML, CSS et JavaScript

Langages serveur

Langage de programmation exécuté sur le serveur pour générer des pages web dynamiques, gérer les bases de données et traiter les requêtes des utilisateurs

Frameworks

Un framework est un ensemble d’outils et de règles qui aide les développeurs à créer plus facilement des sites ou des applications

Bases de données

Une base de données est un endroit où l’on range et organise des informations pour pouvoir les retrouver facilement

Site responsive

Un site responsive est un site qui s’adapte automatiquement à la taille de l’écran, que ce soit sur ordinateur, tablette ou mobile

Application mobile

Une application mobile est un logiciel installé sur smartphone ou tablette, conçu pour exploiter les fonctionnalités spécifiques de ces appareils

Partie 3 - Découvrez le fonctionnement du réseau

Serveurs

Un serveur est un ordinateur spécial qui fournit des services ou des informations à d’autres ordinateurs via un réseau

Adresse IP

Une adresse IP est un numéro unique qui identifie chaque appareil connecté à Internet.

Nom d’hôte et DNS

Un nom d’hôte est un nom facile à retenir pour un site (comme google.com), et le DNS est le système qui traduit ce nom en adresse IP pour trouver le bon serveur

Protocoles

Les protocoles sont des règles qui permettent aux ordinateurs de communiquer entre eux sur un réseau.

Partie 4 - Identifiez les métiers du web

Développeurs

Les développeurs sont les personnes qui écrivent le code pour créer des sites, des applications ou des logiciels.

Product Managers et chefs de projets

Ce sont ceux qui organisent, planifient et suivent le bon déroulement d’un projet informatique.

Designers

Les designers imaginent et créent l’apparence visuelle et l’expérience utilisateur d’un site ou d’une application.

Testeurs

Les testeurs logiciel vérifient que les sites ou applications fonctionnent bien et n’ont pas de bugs.

Techniciens informatique

Le technicien informatique aide à installer, réparer et maintenir les ordinateurs et les réseaux.

Langages de développement web

HTML

Front-End

Squelette du site Web. Il définit la structure de base du contenu, comme les titres, les paragraphes, les images, etc.

CSS

Front-End

Langage qui stylise les pages web (couleurs, tailles, mise en page).

JavaScript

Front-End

Langage qui rend les pages web interactives dans le navigateur.

JavaScript

Back-End

Utilisé via Node.js pour gérer les requêtes serveur, les accès aux bases de données et la logique des applications web.

PHP

Back-End

Langage serveur pour créer des pages web dynamiques et gérer les données..

Java

Back-End

Langage puissant pour créer des applications web robustes et multiplateformes

Python

Back-End

Langage simple et polyvalent, utilisé côté serveur pour le web et l’automatisation

C#

Back-End

Langage Microsoft pour développer des applications web et desktop avec .NET

SQL

Bases de données

Langage pour interroger et gérer les bases de données relationnelles.

Boîte à outils du développeur

  • IDE comme VS Code : Logiciel qui aide à écrire, tester et déboguer du code, tout en un seul outil.
  • Github : Plateforme en ligne pour stocker, partager et collaborer sur des projets de code
  • Librairies / bibliothèques : Collections de code prêt à l’emploi pour ajouter des fonctionnalités à un projet.
  • Frameworks : Cadres de travail qui fournissent une structure pour créer des applications plus rapidement (Angular – JavaScript, Laravel – PHP, Django – Python )
